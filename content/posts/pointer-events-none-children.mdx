Sometimes you want to "disable" all the elements inside a certain element. So we can easly use [`pointer-events: none`](https://developer.mozilla.org/en-US/docs/Web/CSS/pointer-events#Basic_example).

## The problem

What if you want to "disable" only some elements but not the others?

## The solution

You can, surprise surprise, "enable" them (by `pointer-events: auto`) even if its parent has `pointer-events: none`.
