## Highlight the code

Jekyll supplys a tag called `highlight`. It can accept some parameters - what's the code language and whether show lines numbers. [More info](https://jekyllrb.com/docs/liquid/tags/#code-snippet-highlighting)

For example, this code

```html
{% highlight html linenos %}
  <button>do alert</button>
  <script>
    document.querySelector('button').addEventListener('click', () => {
      alert('clicked');
    })
  </script>
{% endhighlight %}
```

Will render

```html
<button>do alert</button>
<script>
  document.querySelector('button').addEventListener('click', () => {
    alert('clicked');
  })
</script>
```
