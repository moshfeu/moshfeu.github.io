To present tag posts count use `tag[1].size`

```ruby
{% for tag in site.tags %}
  <li><a href="/path_to_tags/{{ tag[0] | slugify }}">{{ tag[0] }}</a> <{{tag[1].size}}></li>
{% endfor %}
```

<hr />

To display a `ruby` code but prevent compiler from render it, use `{% raw %}`


```ruby
{% raw %}
  {% for tag in site.tags %}
  ...
  {% endfor %}
{% endraw %}
```
